## NidolphEconomy
Модуль экономики для вашего Discord сервера.

## Возможности
- `!profile` - Показ профиля
- `!admin` - Админ-права
- `!eco` - Мод-права (выдаётся через админ-права)
- `!transfer` - Передача монет
- `!flip` - Подкидывание монетки
- `!bank` - Банк для монет
- `!daily | !work` - Заработок монет
